# School Management System

 -A RESTful backend application built with Spring Boot for managing school operations including students, teachers, courses, scheduling, and academic results.

# Tech Stack

 Java 21
 Spring Boot
 Spring Security
 Spring Data JPA
 Hibernate
 Maven
 Database
 MySQL
 Authentication and Authorization
 JWT (JSON Web Token)

### Core Modules
- **Student Management** - CRUD operations, enrollment, profile management
- **Teacher Management** - Profile, subject assignment, class handling
- **Course & Class Management** - Course creation, class scheduling
- **Enrollment System** - Student course registration and management
- **Academic Records** - Grades, attendance, report cards
- **User Authentication & Authorization** - Role-based access (Admin, Teacher, Student)
- **Dashboard** - Overview with key statistics
